# VRC Sticker Maker (Unofficial Community Tool)

VRChat風の吹き出しスタンプを簡単に作成できるブラウザベースのツールです。
アバターの透過PNGや豊富な絵文字・装飾パーツを自由に組み合わせて、高画質なスタンプを素早く作成できます。

> [!IMPORTANT]
> 本ツールはコミュニティによる有志のプロジェクトであり、**VRChat公式とは一切関係ありません**。

## 🚀 ツールURL
https://amerysa.github.io/VRCStickerMaker/

## ✨ 今回のアップデート内容
- **新デザインUI**: 1080p環境に最適化された3カラムレイアウト。
- **絵文字デコレーション**: 200種類以上の Noto Color Emoji スタンプを搭載。
- **自由なキャンバス操作**: マウスホイールによるズーム、Space+ドラッグによるパン（移動）に対応。
- **回転リセット機能**: アイテムを水平に一瞬で戻せる「回転をリセット」ボタンを追加。
- **安定した保存**: ズームや表示位置に関わらず、常に正確な 512px の範囲をエクスポート。

## 🎨 主な機能
- **VRChat風吹き出し**: 文字数に合わせた自動改行と動的なサイズ調整に対応。
- **マルチ画像アップロード**: 複数の画像を読み込み、自由な順序で配置可能。
- **デコレーション素材**: 星、ハート、キラキラ、そして豊富な絵文字スタンプを標準搭載。
- **高度なレイヤー管理**:
  - ドラッグ＆ドロップによる直感的な並び替え。
  - レイヤーの複製（自動連番ナンバリング付与）。
  - 個別カラー（漫符用）や不透明度のカスタマイズ。
- **プライバシー保護**: 画像処理はすべてクライアント（ブラウザ）内で行われるため、画像が外部サーバーに送信されることはありません。

## 🛠 使用技術
- [Fabric.js](http://fabricjs.com/) - キャンバス操作・オブジェクト管理
- [Google Fonts (Noto Sans JP)](https://fonts.google.com/specimen/Noto+Sans+JP) - メインフォント
- [Google Fonts (Noto Color Emoji)](https://fonts.google.com/specimen/Noto+Color+Emoji) - 絵文字スタンプ
- Vanilla JS / HTML5 / CSS3
